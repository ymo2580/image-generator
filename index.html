Puter 모드 사용 안내

Puter는 무료 AI 사이트예요. 첫 사용 시 한 번만 Puter 인증 창이 뜨고, 그 후부터는 자동으로 작동해요.

  1. 토픽 입력 후 "전체 이미지 생성" 클릭
  2. 처음이면 Puter 인증 창이 뜸 (이메일 입력 또는 게스트로 진행)
  3. 승인하면 이미지 생성 시작
  4. 다음부터는 자동으로 작동

💡 모델별로 스타일이 다르니 여러 개 써보세요.
💡 Flux 모델이 가장 균형 좋아요.
💡 사용량 제한 거의 없어요.

Gemini API 키 발급 가이드

  1. aistudio.google.com/apikey 접속
  2. 구글 계정으로 로그인 (별도 가입 X)
  3. "Create API key" 버튼 클릭 → 키 복사
  4. "Gemini API 키" 칸에 붙여넣기
  5. 토픽 추가 → 스타일·비율 선택 → 생성!

💡 한 번 입력하면 다음부터는 자동으로 채워집니다.
💡 Puter 모드는 API 키 없이 다양한 모델 사용 가능해요.
💡 마음에 드는 이미지는 "아카이브 저장"으로 보관할 수 있어요.
