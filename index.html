Gemini API 키 발급 가이드

  1. aistudio.google.com/apikey 접속
  2. 구글 계정으로 로그인 (별도 가입 X)
  3. "Create API key" 버튼 클릭 → 키 복사
  4. "Gemini API 키" 칸에 붙여넣기
  5. 토픽 추가 → 스타일·비율 선택 → 생성!
