개인 프로젝트 · API 키는 브라우저 localStorage에만 저장되며 외부로 전송되지 않습니다

처음 사용 가이드

  1. aistudio.google.com/apikey 접속
  2. 구글 계정으로 로그인 (별도 가입 X)
  3. "Create API key" 버튼 클릭 → 키 복사
  4. "Gemini API 키" 칸에 붙여넣기
  5. 토픽 추가 → 스타일·비율 선택 → 생성!

💡 한 번 입력하면 다음부터는 자동으로 채워집니다.
💡 토픽은 무제한으로 추가할 수 있어요.
💡 토픽이 많으면 시간이 좀 걸려요 (장당 10~30초).
